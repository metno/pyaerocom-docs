AeroCom variables
-----------------

All AeroCom default variables are available in the file
`variables.ini <https://github.com/metno/pyaerocom/blob/master/pyaerocom/data/variables.ini>`__
which is part of the pyaerocom installation.

Access individual variables
~~~~~~~~~~~~~~~~~~~~~~~~~~~

The default access to these variables is via the instance of the
``pyaerocom.VarCollection`` class that is available via
``pyaerocom.const.VARS``:

.. code:: ipython3

    import pyaerocom as pya
    var_access = pya.const.VARS
    var_access




.. parsed-literal::

    <pyaerocom.variable.VarCollection at 0x7f702db8dc50>



.. code:: ipython3

    var_access.od550aer




.. parsed-literal::

    od550aer
    standard_name: atmosphere_optical_thickness_due_to_ambient_aerosol_particles; Unit: 1



.. code:: ipython3

    var_access.wetso4




.. parsed-literal::

    wetso4
    standard_name: tendency_of_atmosphere_mass_content_of_sulfate_dry_aerosol_particles_due_to_wet_deposition; Unit: kg m-2 s-1
